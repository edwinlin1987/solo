About

My name is Edwin, an aspiring software developer at Hack Reactor. JavaScript Racing is a game I created as a solo project.

I'm a terrible typist, especially when it comes to the heavy syntax required in coding. However, when I'm coding my focus is on building whatever I'm trying to build. Thus, I decided to create JavaScript typing to help myself, and anyone else, get better at typing JavaScript.

JavaScript Typist was built with Angular, HTML, and CSS exclusively. It uses certain functions from the Underscore.js library as a context for practicing JavaScript typing so the user doesn't need to come up with code while practicing.
